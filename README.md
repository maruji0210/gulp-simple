# gulp-simple
[![Build Status](https://travis-ci.org/mur6/gulp-simple.svg?branch=master)](https://travis-ci.org/mur6/gulp-simple)

## 使い方
### インストール
```sh
$ npm install
```
### 実行方法
ブラウザが起動し、src以下がウォッチされ、ブラウザがリロードされます。
```sh
$ ./node_modules/.bin/gulp
```
